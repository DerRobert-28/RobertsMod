# Robert' s Mod

- Mainly decorative blocks of miscellaneous Minecraft mods and versions.
- Buildable with MCreator
- How to use:
  - Install Minecraft Forge 1.20.1
  - Download the archive "Robert's Mod.jar".
  - Put it in the Minecraft mods folder.
  - Have fun!
